📲 홈 화면에 앱 추가하는 방법

문자 혹은 커톡으로 받은 앱 주소를 크롬 주소창에 복사해서 실행합니다. 기도서 화면이 뜨면 왼 쪽 상단에 있는 ☰ 버튼을 누르면 맨 상단에 홈 화면에 설치 메뉴가 보입니다. 이를 실행하면 자동 설치가 시작됩니다. 설치가 다 되었다는 메시지가 보이면 크롬에서 나간 후 앱 아이콘을 찾아서 홈 화면으로 복사해줍니다. (이 앱은 크롬에서 최적화되어 있습니다)

카톡에서 바로 브라우저로 [기도서] 화면을 열었으면 홈 화면에 설치 메뉴가 보이지 않습니다. 이때는 링크 주소를 복사하고, 크롬 브라우저을 열어서 주소창에 붙여두기를 해서 기도서를 다시 열면 홈 화면에 설치 메뉴가 보입니다.

아이폰에서는 PWA 앱을 설치할 수가 없으므로 홈 화면에 바로가기를 만들어 주면 편리하게 사용할 수 있습니다. 사파리 브라우저에서 기도서 주소를 복사해주고 기도서를 실행합니다. 밑에 보면 네모 안에 ↑ 표시 버튼이 있는데 이를 누르면 홈화면에 바로가기 추가가 메뉴 생깁니다. 이를 누르고 추가를 누르면 홈화면에 기도서 사이트로 바로가는 아이콘이 만들어집니다. 다만 아이폰에서는 오프라인 시에는 기도서가 실행되지 않습니다.

💡 홈 화면에 추가된 아이콘은 일반 앱처럼 오프 라인에서도 바로 실행할 수 있습니다.
